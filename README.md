# 1. 프로젝트 개요

프로젝트 개요
현재 금융권은 비대면, 디지털화로 인해 비용 절감을 위한 지점 축소를 진행중입니다. 하지만, 급격한 지점 축소로 인해 지점당 직원수는 증가하여, 증가한 인력을 효율적으로 관리해야 하는 필요성을 인지했습니다.

프로젝트 목적
- 은행, 증권사 PB 직원들의 비어있는 예약 시간대를 활용한 비대면 투자 상담 서비스 제공합니다.
- 기존의 스마트브랜치 서비스를 개선하여 비대면 채널을 더욱 강화합니다.
